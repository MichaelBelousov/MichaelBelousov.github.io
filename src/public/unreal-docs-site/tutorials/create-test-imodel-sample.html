Create a Test iTwin and an iModel from a Bentley provided sample

The quickest way to get access to a cloud hosted iModel is by creating a Test iTwin and an iModel seeded with Bentley provided sample data.

  1. Go to https://developer.bentley.com
  2. Click the Sign In button and sign-in using your Bentley account credentials
    • If you have not already registered, click Register now and complete the registration process.
  3. Click on your user icon and navigate to the My iTwins page
  4. Click the + New button
  5. Give your Test iTwin a name
  6. Click Save
  7. iModels in the newly created test iTwin will be listed (it will not have any iModels)
  8. Click the + New button to create a new iModel
  9. Give your test iModel a name
  10. Select Create iModel from Bentley Sample
  11. Choose one of templates
  12. Click the Save button and wait until your iModel is created
