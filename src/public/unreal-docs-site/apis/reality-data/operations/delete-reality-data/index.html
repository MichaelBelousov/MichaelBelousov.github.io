Reality Data

Download API definition:

OpenAPI v3 (JSON)

Delete reality data

DELETE https://dev-api.bentley.com/realitydata/{id}

Deletes a reality data.

To delete a reality data, it must be associated to only one (or zero) project. If it has more associated projects, first dissociate the projects.

Authentication

Requires Authorization header with valid Bearer token for scope realitydata:modify.

For more documentation on authorization and how to get access token visit OAUTH2 Authorization page.

Rate limits

All iTwin Platform API operations have a rate limit. For more documentation on that visit Rate limits and quotas page.
