This API is a Technical Preview and is available for testing purposes only. Do not use in production.

Packages

Download API definition:
OpenAPI v3 (JSON)
Get package system type properties
GET https://dev-api.bentley.com/edfs/itwins/{iTwinId}/packages/{uniqueName}/systems/{systemType}

Retrieve system type and system type properties for the integration package.

Authentication

Requires Authorization header with valid Bearer token for scope itwin-platform.

For more documentation on authorization and how to get access token visit OAUTH2 Authorization page.

Authorization

User must have the edfs_credmng permission assigned at the iTwin level or be an Organization Administrator for the Organization that owns a given iTwin.
