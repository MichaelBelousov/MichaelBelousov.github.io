Enterprise Data Federation Service (EDFS)
This API is a Technical Preview and is available for testing purposes only. Do not use in production.

Systems

Download API definition:
OpenAPI v3 (JSON)
Update system
PUT https://dev-api.bentley.com/edfs/itwins/{iTwinId}/systems/{id}

Update external system information and its credentials set.

You need to provide the SystemId (unique within the integration package) and the SystemType. You can retrieve them

  • By getting integration package system types

You also need to provide the credentials. These are system properties (different for each system type). You can retrieve them

  • By getting integration package system type properties

Authentication

Requires Authorization header with valid Bearer token for scope itwin-platform.

For more documentation on authorization and how to get access token visit OAUTH2 Authorization page.

Authorization

User must have the edfs_credmng permission assigned at the iTwin level or be an Organization Administrator for the Organization that owns a given iTwin.

Response 401 Unauthorized

This response indicates that request lacks valid authentication credentials. Access token might not been provided, issued by the wrong issuer, does not have required scopes or request headers were malformed.

The user has insufficient permissions for the requested operation.

}
