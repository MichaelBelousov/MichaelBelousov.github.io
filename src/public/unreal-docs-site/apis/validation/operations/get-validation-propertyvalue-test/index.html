OpenAPI v3 (JSON)

Get property value validation test details

GET https://dev-api.bentley.com/validation/propertyValue/tests/{id}

Gets the property value validation test details for the specified test id.

  • Test ids can be discovered by calling GET /validation/propertyValue/tests

Authentication

Requires Authorization header with valid Bearer token for scope validation:read.

For more documentation on authorization and how to get access token visit OAUTH2 Authorization page.
