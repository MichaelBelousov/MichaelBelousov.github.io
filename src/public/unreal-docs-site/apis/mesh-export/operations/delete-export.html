Mesh Export
This API is a Technical Preview and is available for testing purposes only. Do not use in production.

Export

Download API definition:
OpenAPI v3 (JSON)
Delete Export
DELETE https://dev-api.bentley.com/mesh-export/{id}

Deletes an export that is completed or invalid.

Authentication

Requires Authorization header with valid Bearer token for scope mesh-export:modify.

For more documentation on authorization and how to get access token visit OAUTH2 Authorization page.

Request

In
Required?
Description

Description

OAuth access token with scope mesh-export:modify

Yes

Setting to application/vnd.bentley.itwin-platform.v1+json is recommended.

Response

Response 204 No Content

Delete successful

Response 401 Unauthorized

This response indicates that request lacks valid authentication credentials. Access token might not been provided, issued by the wrong issuer, does not have required scopes or request headers were malformed.

json
HeaderNotFound
{
    "error": {
        "code": "HeaderNotFound",
        "message": "Header Authorization was not found in the request. Access denied."
    }
}

Response 404 Not Found

Requested export is not available.

json
{
    "error": {
        "code": "ExportNotFound",
        "message": "Requested export is not available."
    }
}

Response 422 Unprocessable Entity

Invalid or inexistent Id. Also occurs on exports that are in progress.

json
{
    "error": {
        "code": "InvalidMeshExportRequest",
        "message": "Cannot delete exported job.",
        "details": [{
            "code": "InvalidParameter",
            "message": "id is invalid.",
            "target": "id"
        }]
    }
}

Response 429 Too many requests

This response indicates that the user has sent too many requests in a given amount of time.

json
{
    "error": {
        "code": "TooManyRequests",
        "message": "More requests were received than the subscription rate-limit allows."
    }
}

Response headers

Name
Description
retry-after

The number of requests exceeds the rate-limit for the client subscription.
